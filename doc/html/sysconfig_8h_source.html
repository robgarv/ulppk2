45 /*
46  * System configuration utility.
47  * Support system initialization settings (.ini file)
48  * parsing. Establishes variables necessary to:
49  * Support logging and core dump facilities.
50  * Support memory mapped I/O based facilities.
51  *
52  * The following environment variables should be
53  * set prior to running an application that uses
54  * this facility:
55  *
56  * SYSCONFIG_APPNAME -- name of the application, defaults to ulppk_app
57  * SYSCONFIG_DATA_DIR -- path to directory containing app specific data
58  * SYSCONFIG_ETC -- defaults to /usr/local/etc
59  * SYSCONFIG_INI_FILE_NAME -- defaults to ulppkapp.ini
60  * MMDQ_DIR_PATH -- Memory mapped deque directory -- defaults to /var/ulppk
61  * MMPOOL_ENV_DATA_DIR -- memory mapped buffer pool directory --- defaults to /var/ulppk
62  */
