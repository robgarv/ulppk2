Detailed Description

mmfor – Memory Mapped File Of Records

A library module for accessing files organized as arrays of structures.

This is built on the functions and types of mmaptom.c and mmapfile.c

Definition in file mmfor.h.

Typedef Documentation

typedef struct _mmfor_handle MMFOR_HANDLE

A specialization of the MMA_HANDLE structure with additional info necessary to support arrays of records.

typedef struct _mmfor_header MMFOR_HEADER

The first thing written to any of these files is the header.

void *  p 
)

Given a memory mapped file handle and a pointer to an address in the mapped memory region, lock the record for reading. Multiple processes can access during a read lock, but a process atttempting a write lock will be blocked until read locks are released. A write lock will cause this function to block.
