Write to the log(s). Intended to be called from the ULPPK_LOG macro (see ulppk_log.h).

Only log text with text level >= the established log level is printed to the log(s). The log level is set in the .ini file for the calling app (section [logging], variable log_level).If no value was provided, the default if ULPPK_LOG_DEBUG (0) is used.

Parameters
funcName of the calling function
fileFile in which calling function is implemented.
lineLine number at which ulppk_log was called.
text_levelThe logging (text) level of the log write request.
fmtpA printf style format string
...variable number of arguments consistent with the fmtp format string
Returns
non-zero if text_level >= logging level (printed to log).

int ulppk_log_set_logconfig ( int  new_dest,
const char *  ident,
int  options,
int  facility 
)

Set the log destination and if SYSLOG is a destination, sets openlog option and facility codes. Returns previous log destination bit mask.

This function should be called BEFORE calling sysconfig_parse_inifile.

log_dest specifies the log destination and is one of the following values:

  • ULPPK_LOGMODE_CONSOLE 1
  • ULPPK_LOGMODE_SYSLOG 2
  • ULPPK_LOGMODE_ALL 3
Parameters
new_dest(see ulppk_log.h for details)
identIdentify string, usually the app name
optionsSee syslog(3). Meaningless if log_dest is ULPPK_LOGDEST_CONSOLE
facilitySee syslog(3). Meaningless if log_dest is ULPPK_LOGDEST_CONSOLE
