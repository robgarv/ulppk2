size_t ioutils_readln ( int  fd,
char *  buff,
size_t  buffsize 
)

Read a string from a descriptor into the provided buffer. Do not overflow the buffer. Read will be terminated after buffsize-1 characters. On return, string will be null terminated. Line terminator characters ARE NOT preserved! Read stops on EOF or detection of line termination.

Returns
One of the following:
  • Return value > 0 ==> number of bytes read
  • Return value < 0 ==> error
  • Return value == 0 ==> EOF
