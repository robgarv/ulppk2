Detailed Description

Utilities for handling diagnostic error output.

To enable debug trace output, compile with -DULPPK_DEBUG.

Format an error message. Similar to err_msg but provides handling for errno and errno's related error message string.

Parameters
buffpPointer to buffer to receive text. If buffp is NULL, then memory is allocated by the function. This memory MUST be released by calling free. Otherwise, buffp is a buffer of sufficient size provided by the caller, and which will receive the formatted error message.
labelLabel shows up first in the error text
funcName of function in which this function was called.
fileSource file in which function is defined
lineLine at which verr_msg was invoked.
fmtpA format string
apVariable length argument list.
