Detailed Description

Command argument parsing and access functions.

cmadargs provides methods for defining command line option syntax, parsing command line options, and retrieving their values. The mechanism also provides for error reporting and command line help features in a convenient fashion.

The file cmdargs.h contains some useful top level commentary.
