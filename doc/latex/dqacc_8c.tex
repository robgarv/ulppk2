\subsection{Detailed Description}
Double Ended Queue (deque) Access -- dqacc. This package implements a collection of functions by which double ended queues or deques may be defined, initialized, and accessed. A double ended queue consists of a header structure and an array of slots. The size of the array is detemined by the maximum number of items to be pushed into the deque and the size of the item type.
