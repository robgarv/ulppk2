\subsection{Detailed Description}
Memory Mapped Double Ended Queue Support. This is a compound construct that integrates a memory mapped file (see \hyperlink{mmapfile_8c}{mmapfile.\-c}) with a double ended queue (deque) (see \hyperlink{dqacc_8c}{dqacc.\-c}) mmdeques are deques that are persistent and shareable between processes.

All these functions perform locking of the memory mapped deque before operating, and are thus thread safe.

The package reads the environment variable M\-M\-D\-Q\-\_\-\-D\-I\-R\-\_\-\-P\-A\-T\-H. This should be the directory that will contain the deque files. If you are using \hyperlink{sysconfig_8c}{sysconfig.\-c} and ini file support (\hyperlink{ifile_8c}{ifile.\-c}) then this environment variable can be overridden by inifile settings.

See dequetool (\hyperlink{dequetool_8c}{dequetool.\-c}) utility. Dequetool provides features for creating, loading, reading, report on, and resetting memory mapped deques.
