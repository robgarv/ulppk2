\subsection{Detailed Description}
Linked List Access Facility (llacc) This package implements a library of linked list access and manipulation functions. A standardized linked list header format is defined. This format is given in the description of \hyperlink{llacc_8h}{llacc.\-h}, which must be included in the code of any program unit invoking llacc functions.
