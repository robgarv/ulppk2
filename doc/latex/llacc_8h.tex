\subsection{Detailed Description}
This include file contains all definitions necessary for utilitizing the linked list handling package \hyperlink{llacc_8c}{llacc.\-c}. llacc provides function for management and access of linked lists of nodes.

A linked list node contains a pointer to a user defined data buffer.
